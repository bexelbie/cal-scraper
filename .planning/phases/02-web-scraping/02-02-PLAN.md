    **Functions to implement:**

    1. `_clean_text(text: str) -> str`
       - Replace `\xa0` (non-breaking space, U+00A0) with regular space
       - Strip leading/trailing whitespace
       - This handles Pitfall 11 (HTML entities — BS4's .get_text() decodes entities but \xa0 remains)

    2. `_extract_single_event(article) -> Event | None`
       - Takes a BeautifulSoup article Tag
       - Extract title: `article.select_one(TITLE_SELECTOR)`
         - If None: log warning with article classes for debugging, return None
         - Title text: `_clean_text(title_el.get_text())`
         - URL: `title_el.get("href", "")`
       - Extract date: `article.select_one(DATE_SELECTOR)`
         - If None: log warning, return None
         - Date text: `_clean_text(date_el.get_text())`
         - Call `parsed = parse_date(date_text)`
         - If parsed is None: log warning (unrecognized date format), return None
       - Extract venue: `article.select_one(VENUE_SELECTOR)`
         - Venue text: `_clean_text(venue_el.get_text())` if found, else `""` (venue not critical — don't skip)
       - Extract description: `article.select_one(DESCRIPTION_SELECTOR)`
         - Description text: `_clean_text(desc_el.get_text())` if found, else `""` (description not critical)
       - Construct and return:
         ```python
         Event(
             title=title,
             dtstart=parsed.dtstart,
             dtend=parsed.dtend,
             all_day=parsed.all_day,
             venue=venue,
             description=description,
             url=url,
             raw_date=parsed.raw_text,
         )
         ```
       - Critical fields requiring skip: title AND date (per D-02)
       - Non-critical fields (empty string fallback): venue, description

    3. `extract_events_from_html(html: str) -> list[Event]`
       - `soup = BeautifulSoup(html, "lxml")`
       - `articles = soup.select(ARTICLE_SELECTOR)`
       - For each article: call `_extract_single_event(article)`
       - Collect non-None results into list
       - Return list

    4. `extract_all_events(pages: list[str]) -> list[Event]`
       - Concatenate results of `extract_events_from_html(page)` for each page
       - Return combined list
