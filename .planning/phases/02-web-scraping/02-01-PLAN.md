    Create cal_scraper/fetcher.py implementing the paginated HTTP fetcher.

    **Module-level constants:**
    ```python
    BASE_URL = "https://www.moravska-galerie.cz/program/deti-a-rodiny/"
    USER_AGENT = "cal-scraper/0.1 (Czech gallery calendar; +https://github.com/bexelbie/cal-scraper)"
    REQUEST_DELAY = 1.0   # seconds between page fetches (per D-04)
    REQUEST_TIMEOUT = 30  # seconds per request
    FAILURE_THRESHOLD = 0.5  # bail if more than 50% of pages fail (per D-03)
    ```

    **Custom exception:**
    ```python
    class ScrapingError(Exception):
        """Raised when scraping cannot produce usable results."""
    ```

    **Functions to implement:**

    1. `_get_page_url(base_url: str, page: int) -> str`
       - Page 1: return base_url unchanged
       - Page N (N≥2): return f"{base_url}page/{page}/"
       - Ensure base_url ends with "/" before appending

    2. `_discover_max_pages(html: str) -> int`
       - Parse html with BeautifulSoup(html, "lxml")
       - Find `soup.select_one("div.ecs-posts")`
       - If found and has `data-settings` attribute: `json.loads(attr)` and return `settings.get("max_num_pages", 1)`
       - On any failure (missing element, bad JSON): return 1
       - Import: `import json`, `from bs4 import BeautifulSoup`

    3. `fetch_page(url: str, session: requests.Session) -> str | None`
       - `response = session.get(url, timeout=REQUEST_TIMEOUT)`
       - `response.raise_for_status()`
       - Return `response.text`
       - Catch `requests.RequestException` as e: log warning `f"Failed to fetch {url}: {e}"`, return None

    4. `fetch_all_pages(base_url: str = BASE_URL) -> list[str]`
       - Create `session = requests.Session()` with `session.headers.update({"User-Agent": USER_AGENT})`
       - Fetch page 1: `page1_html = fetch_page(_get_page_url(base_url, 1), session)`
       - If page 1 fails: raise `ScrapingError("Failed to fetch first page")`
       - Discover max pages: `max_pages = _discover_max_pages(page1_html)`
       - Initialize: `pages = [page1_html]`, `failures = 0`
       - Loop page 2..max_pages:
         - `time.sleep(REQUEST_DELAY)` before each request
         - `html = fetch_page(_get_page_url(base_url, page), session)`
         - If html is None: `failures += 1`, log warning, continue
         - Else: `pages.append(html)`
       - After loop: if `failures / max(max_pages - 1, 1) > FAILURE_THRESHOLD`: raise `ScrapingError(f"Too many failures: {failures}/{max_pages - 1} pages failed")`
       - Return pages
       - Import: `import time`, `import logging`, `import requests`

    Per D-01: Uses data-settings JSON for pagination discovery (not hardcoded page count).
    Per D-03: Simple retry-free approach — warn on individual failures, bail on majority.
    Per D-04: 1-second delay, descriptive User-Agent.
    Per SCRP-06: requests.RequestException caught broadly (covers Timeout, ConnectionError, HTTPError).
